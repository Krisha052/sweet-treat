Vixien’s Soda Cans
Pixel Art Asset Pack

This pack contains pixel art soda can sprites designed for 2D games,
inventories, props, and world items.

CONTENTS
- 64 soda can designs
- 4 directions per can (front, right, back, left)
- 4 top variants
- 1 bottom sprite
- Total sprites: 261

RESOLUTION
- Side views: 16x32 pixels
- Top views: 16x16 pixels
- Bottom view: 16x16 pixels

FORMATS
- PNG (transparent background)
- Aseprite source files included

SPRITE SHEETS
Two sprite sheet versions are included:
- Padded sheets for editing and preview
- Unpadded sheets for direct use as tilemaps

PALETTE
Palette credit and link are provided in PaletteInfo.txt
